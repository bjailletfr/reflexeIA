Guide d'évaluation Réflexe IA

Ce document vous aide à remplir votre bilan éthique :

  • Compte élève : Pas de collecte de données personnelles directes.
  • Âge : Respect strict des conditions d'utilisation de l'éditeur.
  • Frugalité : Privilégier les modèles moins gourmands en calcul.
  • Vérifier : Toujours croiser les sources produites par l'IA.
  • Transparence : L'élève doit savoir qu'il interagit avec une machine.
  • Plus-value : L'IA doit libérer du temps ou permettre une tâche impossible autrement.
  • Honnêteté : Citer l'usage de l'IA dans le rendu final.
  • Expertise : L'enseignant reste le seul juge de la validité académique.
  • Libre : Favoriser les outils "Open Source" quand c'est possible.
